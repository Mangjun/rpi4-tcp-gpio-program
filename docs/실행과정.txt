<원격 장치 제어 시스템 실행 및 사용 가이드>

- 실행 전 준비 사항 -
프로젝트 최상위 디렉토리에서 scp.sh 파일을 열어 서버 접속 정보(사용자명, IP, 홈 디렉터리 경로)를 본인의 라즈베리파이 환경에 맞게 반드시 수정해야 합니다.

- 프로그램 실행 순서 -
스크립트 수정이 완료되면 아래 명령어를 순서대로 실행하여 시스템을 구동합니다.

[1] 자동 빌드 및 배포 (프로젝트 최상위 폴더에서 실행)
./scp.sh

[2] 서버 실행 (라즈베리파이4 홈 디렉터리에서 실행)
./server

[3] 리눅스 클라이언트 실행 (리눅스 프로젝트 루트 디렉터리에서 실행)
./bin/client <서버_IP> <서버_PORT>

리눅스 클라이언트 (CLI) 제어 메뉴
클라이언트가 정상적으로 서버에 접속되면, 아래의 메뉴 번호를 입력하여 각 하드웨어 장치를 제어할 수 있습니다.

1번 (LED): ON, OFF, BLINK, LOW(최저 밝기), MID(중간 밝기), HIGH(최대 밝기)

2번 (BUZZER): ON, OFF

3번 (CDS): ON, OFF (※ 켜져 있을 시 임계값 수정 가능)

4번 (SEGMENT): ON (※ 실행 후 출력할 타이머 숫자값 입력)

5번 (프로그램 종료): 클라이언트 프로그램 안전 종료

웹 클라이언트 접속 안내
CLI 환경 외에도, 웹 브라우저를 통해 직관적인 대시보드 UI로 장치를 모니터링하고 제어할 수 있습니다.

접속 주소: http://<서버_IP>:8080/

특징: 별도의 새로고침 없이도 다른 클라이언트나 센서에 의한 장치 상태 변화가 실시간으로 UI에 동기화되어 쉽게 확인할 수 있습니다.
